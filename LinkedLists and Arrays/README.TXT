A combination of arrays and linked lists to form a growable linked array. The growable array is a linked list where the contents of each node of the list are an array of elements, rather than just a single element as in a standard linked list.
------------------------------------------------------------------------------------------------------------------------------------------

PROJECT TITLE: Combined Data Structures
PURPOSE OF PROJECT: Implementing a growable data structure.
